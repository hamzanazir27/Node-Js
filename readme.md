## 🚀 Getting Started

### Installation & Setup

```bash
# Clone the repository
git clone https://github.com/hamzanazir27/Node-Js.git

# Navigate to the project directory
cd Node-Js

# Install dependencies (for any specific module)
npm install

# Start with the basics
cd 01-introduction
node index.js

# Or jump to any completed module
cd 29-blog-project-complete
npm install
npm start
```

### Quick Module Navigation

```bash
# Recent completions (recommended to explore)
cd 33-nodejs-clustering        # Latest: Scalability patterns
cd 32-nodejs-streams          # Performance optimization
cd 31-web-sockets            # Real-time applications
cd 29-blog-project-complete  # Full-stack blog application

# Core learning modules
cd 18-mvc-pattern            # Architecture patterns
cd 21-authentication         # JWT implementation
cd 16-mongodb-database       # Database fundamentals
```

### Environment Requirements

- **Node.js**: v14+ (Chrome V8 Engine)
- **MongoDB**: Local or Atlas connection
- **npm**: Latest version
- **Postman**: For API testing (optional)

### Running Projects

```bash
# Blog Application (Complete Project)
cd 29-blog-project-complete
npm install
npm start
# Visit: http://localhost:3000

# URL Shortener Service
cd 19-url-shortener-project
npm install
node server.js

# WebSocket Chat
cd 31-web-sockets
npm install
node app.js
```
